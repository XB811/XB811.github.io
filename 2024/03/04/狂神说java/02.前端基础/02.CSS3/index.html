    4. 父类添加一个伪类：after

    小结;

    1. 浮动元素后面增加空div

      简单，代码中尽量避免空div

    2. 设置父元素的高度

      简单，元素设置了固定的高度，就会被限制

    4. 父类添加一个伪类：after（推荐）
